COMP 2156 DevOps course teaches students the procedures, protocols, and best practices used by DevOps engineers to improve workplace productivity. Where, students learn to analyze, evaluate, and implement DevOps solutions using industry-standard tools and hands-on exercises. The data formats taught to students in this class include, XML, JSON, YAML, and YANG, as well as DevOps tools such as Ansible and Git. Which are crucial for network automation and software deployment.
